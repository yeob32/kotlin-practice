# 코틀린

## 코틀린 ?

요약
- 코틀린은 타입 추론을 지원하는 정적 타입 지정 언어이다.
- 객체지향과 함수형 프로그래밍 스타일 모두를 지원한다.
- 서버 애플리케이션, 안드로이드 등 다양한 대상한 플랫폼에서 활용 가능하다.
- 코틀린은 무료이며 오픈소스이다. 주요 IDE와 빌드 시스템을 지원한다.
- 코틀린은 실용적이며, 안전하고, 간결하며 상호운용성이 좋다.
